Eksplorasi Machine Learning & Kecerdasan Buatan
Selamat datang di repositori dokumentasi belajar saya. Proyek ini merupakan wadah implementasi praktis dari konsep-konsep Machine Learning yang saya pelajari melalui berbagai literatur teknis.

📌 Fokus Pembelajaran
Repositori ini mencakup berbagai latihan dan proyek kecil yang berfokus pada:

Dasar Pemrosesan Data: Implementasi teknik Encoding dan Decoding pada data tekstual maupun numerik.

Arsitektur Model: Pemahaman mendalam mengenai struktur algoritma, terinspirasi dari konsep dasar Neural Networks dan Transformer.

Implementasi Praktis: Penerapan model pada dataset dunia nyata untuk analisis prediktif.

🛠️ Alat & Teknologi
Untuk memastikan efisiensi dan reproduksibilitas kode, saya menggunakan stack teknologi berikut:

Environment: Jupyter Lab & uv sebagai manajer paket yang efisien.

Library Utama: Pandas, NumPy, dan pustaka pendukung Deep Learning lainnya.

📖 Referensi Utama
Proyek dalam repositori ini dibangun berdasarkan panduan dari sumber-sumber terpercaya, di antaranya:

Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow oleh Aurélien Géron.

Seri pembelajaran Neural Networks oleh Andrej Karpathy dan Bapak Made Satria Wibawa, Ph.D.
